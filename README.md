# 💡 CFL Bulb – SolidWorks 3D Model

A detailed 3D model of a **Compact Fluorescent Lamp (CFL) Bulb** designed using **SolidWorks**. This project demonstrates the use of advanced SolidWorks features to recreate a realistic CFL bulb geometry.

## ⚙️ Features Used

| Feature | Purpose |
|--------|---------|
| `Revolve1` | Rotates the base profile to create the bulb cap |
| `Boss-Extrude1` | Builds the main solid body of the base |
| `Dome1 & Dome2` | Adds rounded geometry to the top and bottom |
| `Helix/Spiral1 & 2` | Defines the path for the CFL spiral tubes |
| `Cut-Sweep1` | Hollows out the tube along the helix path |
| `Sweep1` | Creates the CFL tube body along the spiral |
| `Fillet1` | Smooths edges for a realistic appearance |

---

## 🧠 What I Learned

- Creating complex **helix and spiral paths** for sweep operations
- Using **Dome features** for organic rounded surfaces
- Combining **Helix + Sweep** to model real-world tube geometry
- Applying **Fillet** to achieve smooth, production-ready edges
- Managing **multiple solid bodies** in a single part file
